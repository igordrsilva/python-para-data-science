# 🐍 Python for Data Science

## 🇺🇸 About this repository
This repository contains practical notebooks developed during my Data Science learning journey, focusing on:
* Exploratory Data Analysis (EDA)
* Data visualization
* Data cleaning and understanding
* Analytical thinking applied to real datasets

The goal is to build strong fundamentals and demonstrate technical skills used in real-world data analysis.

## 📊 What you will find here
🔹 Exploratory Data Analysis (EDA)
* Dataset structure inspection
* Missing values detection
* Descriptive statistics
* Pattern identification

🔹 Main Methods for Initial Data Analysis. Core methods applied using Pandas:
* .head() / .tail()
* .shape
* .info()
* .describe()
* .nunique()
* .value_counts()
* .mean(), .median(), .mode()
* .std()
* .dtypes
* .columns

🔹 Data Visualization
* Distribution analysis
* Pattern recognition
* Trend identification
* Insight communication

Libraries used:
* Matplotlib
* Seaborn

## 🚀 How to Run
1️⃣ Clone the repository
```
git clone https://github.com/igordrsilva/python-para-data-science.git
```

2️⃣ Navigate to the folder
```
cd python-para-data-science
```

3️⃣ Install dependencies
```
pip install pandas numpy matplotlib seaborn jupyter
```

4️⃣ Start Jupyter Notebook
```
jupyter notebook
```
